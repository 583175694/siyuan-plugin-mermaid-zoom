# Changelog

## v0.1.0 (2026-02-06)

### Features

- Double-click Mermaid/Graphviz/PlantUML diagrams to open fullscreen preview
- Mouse wheel zoom with cursor-centered scaling
- Drag to pan within the preview dialog
- Reset zoom button
- Export diagram as PNG (2x resolution)
- Keyboard shortcut (Alt+Shift+Z) to open preview
- Hover zoom button on diagram blocks
- Right-click menu integration
- i18n support (English & Chinese)
